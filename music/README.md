# BGM 音樂檔案

請將背景音樂檔案命名為 `bgm.mp3` 並放置在此目錄。

## 推薦音樂來源

**Pixabay 免費音樂** (CC0 授權)

下載步驟：
1. 前往 https://pixabay.com/music/search/chinese%20new%20year/
2. 選擇喜歡的中國新年音樂（推薦：Chinese New Year 相關曲目）
3. 點擊音樂進入詳情頁
4. 點擊「Free Download」按鈕下載 MP3
5. 將下載的檔案重新命名為 `bgm.mp3`
6. 放置到此 `music/` 目錄

## 支援格式

瀏覽器支援：`.mp3`、`.ogg`、`.wav`

## 注意事項

- 若缺少此檔案，遊戲仍可正常運行，只是沒有背景音樂
- 瀏覽器要求使用者必須先有互動（點擊）才能播放音訊，屬正常行為
- 若 console 出現 `[BGM] 播放失敗` 訊息，請確認檔案路徑正確
